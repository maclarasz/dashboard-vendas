# 📊 Dashboard de Vendas 2024

Dashboard interativo para acompanhamento de vendas mensais, 
metas e desempenho por categoria de produto.

## 🔗 Demo ao vivo
[Clique aqui para acessar o dashboard](https://dashboard-vendas-s4q9h4sbrlpujxfjrtwmry.streamlit.app)

## 💡 Funcionalidades

- Métricas de total de vendas, melhor mês e meta anual
- Gráfico de linha comparando vendas vs meta por mês
- Gráfico de barras de vendas por categoria
- Tabela completa com todos os dados

## 🛠 Tecnologias

- Python
- Streamlit
- Pandas
- Plotly

## 🚀 Como rodar localmente

1. Clone o repositório
2. Instale as dependências:
   pip install -r requirements.txt
3. Rode o app:
   streamlit run app.py
